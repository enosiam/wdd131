Project overview

This site was built for the final project. It demonstrates HTML, CSS, and JavaScript skills, responsive design, accessibility features, and dynamic functionality (filters, favorites, localStorage, and form handling).

Accessibility & usability

  • Semantic HTML (landmarks, headings, lists)
  • Keyboard-accessible controls and visible focus states
  • Color contrast tested and conservative palette
  • All images include descriptive alt text

Design choices

Fonts: Poppins (weights 400 & 600). Colors: Navy for primary accents and gold highlights. Card-based layout for easy scanning and responsive images with lazy loading.
